Comment utiliser eXo Platform 3 sur votre Android


eXo Platform - Version 1.0


Page de connexion


Connectez-vous: vous devez saisir l'URL de votre instance de eXo Platform est située, avec votre nom d'utilisateur et mot de passe.




Paramètres


Touch sur le bouton Paramètres pour voir réglage de la langue de dialogue



Écran principal


Après avoir terminé la connexion, l'application déplace l'utilisateur à l'écran principal



Files


Une liste des fichiers disponibles dans votre eXo Platform référentiel apparaîtra après avoir cliqué sur l'icône des fichiers.



Pour renommer, supprimer, copier, couper ou coller un fichier, sélectionnez l'option "action" icône (la flèche bleue) à côté du nom de l'élément.


Télécharger le fichier en SDCard


Chat


Une liste des autres utilisateurs à la disposition de chat apparaît après avoir cliqué sur l'icône de Chat. Sélectionnez un utilisateur, et une fenêtre de discussion est ouvert.



Touch sur chaque élément de l'utilisateur dans la liste pour passer à la fenêtre de chat


Dashboard


Tout gadgets que vous avez déjà ajouté à votre tableau de bord eXo Platform apparaîtra ici, comme toutes les personnalisations ou les modifications futures à votre tableau de bord.



Il suffit de toucher le nom du gadget pour afficher un élément.
