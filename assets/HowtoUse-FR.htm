Comment faire pour utiliser


eXo Platform - Version 1.0


Page de connexion


Afin de se connecter à la plate-forme, vous devez entrer une plate-forme du nom d'hôte, nom d'utilisateur et mot de passe avant de toucher le bouton Connexion




Paramètres


Touch sur le bouton Paramètres pour voir réglage de la langue de dialogue



Écran principal


Après avoir terminé la connexion, l'application déplace l'utilisateur à l'écran principal



File Application


Touch sur les fichiers icône pour démarrer l'application fichiers



Vous pouvez effectuer des actions sur les fichiers et dossiers en appuyant sur le bouton actions


Pour afficher le contenu d'un fichier, l'utilisateur a besoin de toucher sur chaque élément dans la liste pour télécharger le fichier et l'afficher à l'aide de n'importe quel explorateur application


Chat Application


Touch sur l'icône Chat de démarrer l'application Chat



Touch sur chaque élément de l'utilisateur dans la liste pour passer à la fenêtre de chat


Gadget


Touch sur le point d'entrer dans le tableau de bord Liste Gadget



Touch sur un point Gadget pour commencer
