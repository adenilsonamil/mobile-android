How to Use


eXo Platform - Version 1.0 on your Android


Log In page


Login: you must enter the URL where your instance of eXo Platform is located, along with your existing username and password




Setting


Touch on the Setting button to show Language Setting dialog



Main Screen


After finishing the login progress, application will move user to the Main Screen



File Application


A list of available files in your eXo Platform repository will appear after tapping the Files icon



To rename, delete, copy, cut or paste a file, select the "action" icon (the blue arrow) beside the item name.


Download file into SDCard


Chat Application


A list of other users available to chat will appear after tapping the Chat icon.



Select a user, and a chat window will open.


Dashbard


Any gadgets you have previously added to your eXo Platform dashboard will appear here, as will any customizations or future changes to your dashboard.



Simply touch the gadget name to view an item.
